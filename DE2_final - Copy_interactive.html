Câu 22. Trong không gian $Oxyz$ , cho hai điểm $A\left( {2;0;1} \right)$ và $B\left( { - 1; - 6;7} \right)$ và mặt phẳng $\left( P \right):x - y + z - 3 = 0$ . Xét điểm $M$ thuộc mặt phẳng $\left( P \right)$ , tìm giá trị nhỏ nhất của $2M{A^2} + M{B^2}$ .

Lời giải câu 22:
Gọi điểm $I$ thỏa mãn $2\overrightarrow {IA} + \overrightarrow {IB} = \overrightarrow 0$ .
$\Rightarrow I\left( {1; - 2;3} \right)$ .
$2M{A^2} + M{B^2} = 2{\left( {\overrightarrow {MI} + \overrightarrow {IA} } \right)^2} + {\left( {\overrightarrow {MI} + \overrightarrow {IB} } \right)^2}$ $= 3M{I^2} + 2\overrightarrow {MI} \left( {2\overrightarrow {IA} + \overrightarrow {IB} } \right) + 2I{A^2} + I{B^2}$
$= 3M{I^2} + 2I{A^2} + I{B^2} = 3M{I^2} + 54$ .
Để $2M{A^2} + M{B^2}$ nhỏ nhất thì $M$ là hình chiếu của $I$ lên mặt phẳng $\left( P \right)$ .
$\Rightarrow I{M_{\min }} = d\left( {I,\left( P \right)} \right) = \dfrac{\left | {1 - \left ( { - 2}\right ) + 3 - 3}\right |}{\sqrt{{1^2} + {{\left ( { - 1}\right )}^1} + {1^2}}} = \sqrt 3$ .
Vậy $2M{A^2} + M{B^2}_{\min } = 3{\left( {\sqrt 3 } \right)^2} + 54 = 63$ .
